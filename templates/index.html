RALEIGH CRIME MAP
Users can utilize this map to analyze relative danger levels of neighborhoods within the Raleigh city limits. This application is ideal for people looking to judge the safety of their current or future home/work environments. Users can also see trends in criminal activity for the neighborhood in question by viewing the 12 month historical crime index chart per neighborhood.

SERIAL CRIME MAP
The Serial crime map is an example of what can be done with data that has both time and geographic components. The data used to drive this application comes from a 15 year old homicide case, covered in the popular podcast Serial. The application very easily allows the user to explore a vast data set of testimonies and interviews on a beautiful map with an interactive timeline of events.

COACH K 1K TWEET SCRAPER
This project was built to commemorate Duke Basketball coach Coach Mike Krzyzewski's career. This site collected live Twitter data during the game that gave Coach K his 1000th win.
